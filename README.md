# AppHiper — Migración a PWA (Next.js) + API (NestJS)

Monorepo inicial con frontend (Next.js PWA) y backend (NestJS + Prisma).

Estructura creada:

- `frontend/` — Next.js PWA skeleton
- `backend/` — NestJS API skeleton con Prisma schema

Siguientes pasos:

1. Instalar dependencias en `frontend` y `backend`.
2. Configurar variables de entorno en `backend/.env`. Si usas Supabase, copia el `DATABASE_URL` del proyecto Supabase.
3. Ejecutar en modo desarrollo.

See the individual READMEs in each folder for commands.
